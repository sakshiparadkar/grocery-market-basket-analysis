Market Basket Analysis — Groceries Dataset

This project analyzes 38,000+ grocery transactions to discover products that customers frequently buy together using the Apriori algorithm and Association Rule Mining.

Using Python libraries like pandas, mlxtend, matplotlib, seaborn, and networkx, the analysis identifies hidden shopping patterns based on Support, Confidence, and Lift metrics.

The project includes multiple visualizations such as top purchased items, basket size distribution, association rules, heatmaps, and network graphs to clearly show relationships between products.

These insights can help retailers improve product placement, cross-selling strategies, and promotional bundles to increase sales.
